### 插值
*	文本：{{}}
*	html：v-hml
*	属性：v-bind

### 指令
*	v-if：是否插入
*	v-show：是否显示
*	v-bind：更新属性
*	v-on：监听事件
*	v-model：双向数据绑定

### 过滤器
*	函数，串联，参数

### 缩写
*	v-bind: -> :
*	v-on: -> @

### v-for
*	迭代对象：value key index
*	迭代整数

### 计算属性
*	computed vs methods：computed使用缓存，性能更好，但是如果你不希望缓存，你可以使用 methods 属性

### 样式绑定
*	class属性绑定
*	style(内联样式)
